## Model comparison and notes

**What happened to Gemini this run.** The Gemini key in `.env.local` had its prepaid credits
depleted at the start of the run (`You exceeded your current quota` / `prepayment credits are
depleted`). So the v4 set was completed on **fal Flux** (`fal-ai/flux/dev`, via `FAL_KEY`), tagged
honestly. The before/after pairs were dropped for v4 (fal text-to-image cannot take the flat-lay as
an image input for true garment correspondence); the **verified v3 Gemini before/after pairs**
(in `public/before-after/`) already satisfy that requirement and are surfaced on `/design`.

**The comparison we can still make** (fal Flux v4 vs the v3 Gemini Nano Banana Pro set, both shown
side by side on `/design`):

| Use case | fal Flux (v4) | Gemini Nano Banana Pro (v3) |
| --- | --- | --- |
| Editorial figure (hero/gallery) | Strong. Clean chiaroscuro, good hands, cohesive warm grade, on-brand oxblood/cream. | Strongest. Slightly finer skin micro-texture and fabric/embroidery detail. |
| Indian heritage (saree, lehenga) | Very good, convincing drape and jewellery. | Excellent, marginally richer fabric realism. |
| Texture / background plates | Excellent and cheapest; clean seamless plates. | Overkill for flat plates. |
| Before/after correspondence | Not possible (no image input on text-to-image). | The right tool (image input keeps the exact garment). |
| Cost per image | ~$0.025 | ~$0.134 |

**Verdict / recommendation.** fal Flux is a genuinely strong, ~5x cheaper option and is the right
default for textures, backgrounds, and preset thumbnails; it is also good enough for editorial
figures. For the hero and gallery "money shots" and for any before/after, **Gemini Nano Banana Pro
remains the quality leader and the only option for image-correspondence** — regenerate those on
Gemini once the key is topped up (`npx tsx scripts/generate-assets-v4.ts`, with the gemini-tagged
assets; drop `GEMINI_TO_FAL`). 

**Actual spend this phase: ~$0.58** (35 fal Flux images at ~$0.025; Gemini ~$0.00 because the key
was depleted). This is far under the $20-45 budget because fal is cheap and Gemini produced nothing
this run. Topping up Gemini and regenerating the ~20 editorial money-shots would add roughly
$2.50-3.00.
